This is a PHP based web application development framework created by Ashwini Rath. Batoi Open Source Framework (OSF) encompasses the multi-tier architecture of web, and enables entreprise-grade web application development really fast and easy. for more details about the framework, please visit its home page at https://framework.batoi.com

WHAT THIS DOWNLOAD CONTAINS
----------------------------
1. OSF IDE (/osf/ide)
2. OSF Libraries (/osf/lib)
3. OSF system files
4. Application file structure and sample files

For more details about file structure, please visit https://www.batoi.com/resources/docs/framework/batoi-framework-application-file-structure

TECHNOLOGIES
-------------
> PHP7.3 and any database supported by PHP Data Object (PDO)
